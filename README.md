# 第1天 核心成果：从混乱到规范

**解决的问题**：客户数据中因手工录入导致数字与文本（如“一百”）混合，无法直接计算分析。

**我的方案**：
1 使用 Power Query 的【替换错误】功能，将错误值批量转换为`0`。
2.  将此清洗过程保存为可重复使用的流程。

**创造的价值**：
1.  **数据保护**：绝不修改客户原始文件。
2.  **自动化**：客户未来只需 **右键点击 → 【刷新】**，即可基于新数据自动获得干净报表。
3.  **效率提升**：将数小时的手工检查，变为几秒钟的自动操作。


> **核心认知**：今天我构建的不是一个静态表格，而是一个 **“可重复使用的数据清洗流水线”**。
